07.2 — Semantic Color Modifiers

Add to any type class. Color modifiers use !important to override role defaults.

metui-text--seed metui-text--cyan metui-text--green metui-text--red metui-text--yellow metui-text--muted metui-text--subtle

07.3 — Line Height & Reading Density
Body text · lh 1.7

Standard body copy at 1.7 line-height. Generous spacing prevents the "wall of text" common in legacy terminals.

Use .metui-body-lg for primary documentation and .metui-body-sm for secondary supporting copy.

/* --- Display --- */
.metui-display-lg  { font-size:2.0rem; font-weight:700; text-transform:uppercase; color:var(--seed); letter-spacing:2px; line-height:1.1; }
.metui-display-sm  { font-size:1.4rem; font-weight:700; text-transform:uppercase; color:var(--seed); letter-spacing:1px;  line-height:1.2; }
.metui-mono-display{ font-size:2.4rem; font-weight:700; color:var(--seed); line-height:1; letter-spacing:-1px; }

/* --- Headline --- */
.metui-headline-lg { font-size:1.2rem; font-weight:700; color:var(--fg); border-bottom:1px solid var(--seed); padding-bottom:4px; line-height:1.3; }
.metui-headline-sm { font-size:1.0rem; font-weight:700; color:var(--fg); line-height:1.3; }

/* --- Title --- */
.metui-title-md    { font-size:0.9rem;  font-weight:700; color:var(--cyan); line-height:1.4; }
.metui-title-md::before { content:"[ "; }
.metui-title-md::after  { content:" ]"; }
.metui-title-sm    { font-size:0.82rem; font-weight:700; color:var(--fg);   line-height:1.4; }

/* --- Body --- */
.metui-body-lg     { font-size:0.9rem;  font-weight:400; color:var(--fg);       line-height:1.7; }
.metui-body-sm     { font-size:0.82rem; font-weight:400; color:var(--fg-muted); line-height:1.7; }

/* --- Label --- */
.metui-label-md    { font-size:0.72rem; font-weight:700; text-transform:uppercase; letter-spacing:2px; color:var(--fg-subtle); line-height:1.4; }
.metui-label-sm    { font-size:0.65rem; font-weight:400; text-transform:uppercase; letter-spacing:2px; color:var(--fg-subtle); line-height:1.4; }
.metui-label-sm::before { content:"// "; }

/* --- Inline code --- */
.metui-code        { font-size:0.82rem; color:var(--yellow); background:var(--surface-2); padding:2px 6px; }

/* --- Color modifiers --- */
