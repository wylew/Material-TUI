11.3_A11Y_AND_FALLBACKS

Never rely solely on color (e.g., Green for Success). Always include a Textual Indicator.

  • [OK] Process_Success
  • [ER] Connection_Failed
  • [WR] Memory_Threshold
11.4_MOTION_AND_RHYTHM

In a TUI, motion should be Functional, not Decorative.

  • Blink: Use only for the active text insertion point (The Cursor).
  • Pulse: Use only for critical system halts or empty states.
  • Step: Use for state transitions to maintain the "Hardware" feel.
