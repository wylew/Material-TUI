# 📟 METUI (Material Expressive TUI)
### *A High-Density, MD3-Compliant Terminal User Interface Library*

**METUI** bridges the gap between the structured logic of **Material Design 3** and the raw, high-contrast aesthetic of **Legacy Terminal Interfaces**. It is designed for developers who want the "Power User" feel of a CLI with the accessibility and motion of a modern design system.

---

## ⚡ Quick Start

To integrate the METUI design system, include the **JetBrains Mono** font and apply the core CSS variables to your project root.

```html
<link href="[https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;700&display=swap](https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;700&display=swap)" rel="stylesheet">

<div class="box">
    <span class="box-title">SYSTEM_STATUS</span>
    <button class="glow-focus">[ EXECUTE_COMMAND ]</button>
</div>
